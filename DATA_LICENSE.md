# 데이터 라이선스

Korea AI Map의 **데이터**(`data/` 및 빌드 산출물 `web/public/data/*.json`)는
[Creative Commons Attribution 4.0 International (CC BY 4.0)](https://creativecommons.org/licenses/by/4.0/) 하에 배포됩니다.

## 요약

- **자유롭게** 공유·재배포·수정·상업적 이용 가능
- **조건**: 출처 표시 — 프로젝트명(**Korea AI Map**)과 데이터 버전을 표시해야 합니다.

예시 표기:
> 출처: Korea AI Map (버전 YYYY.MM), https://github.com/korea-ai-map/korea-ai-map.github.io, CC BY 4.0

## 범위 구분

| 대상 | 라이선스 |
|---|---|
| 구조화 데이터 (`data/`, `web/public/data/`) | CC BY 4.0 (또는 ODC-BY) |
| 문서 (`*.md`, `docs/`) | CC BY 4.0 |
| 코드 (`web/`, `scripts/`, `schemas/` 생성기) | [MIT](LICENSE) |
| 각 조직의 로고·상표 | 각 권리자에게 귀속 (본 라이선스 적용 대상 아님) |

## 면책

데이터는 공개된 출처를 기반으로 정리되며 정확성을 보장하지 않습니다.
오류는 [이슈](../../issues)로 제보해 주세요. 등재는 추천·인증·투자 의견을 의미하지 않습니다.
